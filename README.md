Projet "Cinetech"

Le but de ce projet est de réaliser une bibliothèque de films et de séries, en utilisant l'API de TMDB.

Pour lancer ce projet:

- Clonez le repo (git clone https://github.com/cosmin-bilga/cinetech.git)
- Installez python version 3 (https://www.python.org/downloads/)
- Lancez le serveur avec python en vous mettant à la racine du projet (python -m http.server 8000)
- Accedez le site sur le navigateur (http://localhost:8000)

Si vous voulez modifier le projet:

- Clonez le repo (git clone https://github.com/cosmin-bilga/cinetech.git)
- Installez typescript (npm install -g typescript)
- Lancez le compilateur typescript en mode watch (npx tsc --watch)
- Installez le CLI Tailwind (npm install tailwindcss @tailwindcss/cli)
- Lancez le watcher de tailwind (npx @tailwindcss/cli -i index.css -o ./dist/output.css --watch)
